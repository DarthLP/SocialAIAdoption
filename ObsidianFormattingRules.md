# Obsidian Formatting Rules

## Purpose
Reference style file for AI-generated and human-authored vault notes.

## Rules
- Use one `#` heading per note.
- Use `##` and `###` for structured sections.
- Use wikilinks for internal note references.
- Prefer concise bullets and stable heading names.
- Use fenced code blocks with language tags.
- Keep metadata in frontmatter where templates require it.
